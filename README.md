# Hackathon-Degree-Planner

**Elevator Pitch:** UBC Degree Planner: Stop manually cross-checking requirements. Build course schedules, track progress, and graduate on time with AI-powered recommendations.

## Tired of having to cross check degree requirements when registering for courses?

We are too, so we made a smart planner for UBC students to keep track of degree requirements and make sure you graduate ASAP rocky.

### Features

First, select faculty and program. Then, start building your course schedule for each term and track your degree progress.

## Getting Started

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser
